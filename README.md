# telegram-html-escape

### Installation

Paste in terminal and run:  
```bash
npm i telegram-html-escape
```

After that import and use this module as shown:  
```JavaScript
var he = require('./index.js');
console.log(he.esc("/ escape & @ && ? \" )) # < >")); // / escape &amp; @ &amp;&amp; ? &quot; )) # &lt; &gt;
```
